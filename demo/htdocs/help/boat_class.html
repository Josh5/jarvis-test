Introduction

This screen allows any valid user to maintain the list of boat classes. Every boat in the database must belong to a single boat class.

Grid Columns

Mandatory fields are marked in bold.

ClassA text string for the name of the class.
Active?A one character string which is either "Y" or "N". The value of this flag does not have any functional impact.
DescriptionThe class description field is a multi-line text field accessible by clicking the small [+] symbol in the leftmost column. This field is optional, and may be left empty.

Extra Features

Use the "Boats" button to proceed to the boats screen. If you have a boat class selected when you press this button, then that class will be used as a display filter for the "Boats" screen.
