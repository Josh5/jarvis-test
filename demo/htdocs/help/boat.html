Introduction

This screen allows any valid user to maintain the list of boats known to the system.

Grid Columns

Mandatory fields are marked in bold.

ClassDenotes the single boat class to which this boat belongs.
NameA text string for the name of the boat. The name must be unique within the boat class.
Reg #An optional registration number for the boat. No uniqueness constraints are enforced.
OwnerAn optional owner's name for the boat.

Extra Features

Use the "Classes" button to return to the boat classes screen.

Use the "Edit Details" button to edit the additional details for the currently selected boat.
