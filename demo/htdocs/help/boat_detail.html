Introduction

This screen allows the editing of the full set of details for a single boat. In fact, because this is such a simple application, most of the boat details were editable in the in-place grid editing on the main "Boats" screen.

Summary Fields

Mandatory fields are marked in bold.

ClassDenotes the single boat class to which this boat belongs. The class of a boat may not be changed on this screen.
NameA text string for the name of the boat. The name must be unique within the boat class.
Reg #An optional registration number for the boat. No uniqueness constraints are enforced.
OwnerAn optional owner's name for the boat.

Description Fields

Mandatory fields are marked in bold.

DescriptionOptional multi-line description field for this boat. Enter any additional information you wish to record for this boat.

Extra Features

Use the "Boats" button to return to the boats screen.
