Welcome

Introduction

This is a simple demonstration application showing how to combine the back-end web-service provided by Jarvis with a front-end Rich Internet Application.

For information on how to configure this demonstration, please refer to the specific chapter in the Jarvis Guide.

Details

The demo application gives access to a simple "Boat Management" database, which is stored in SQLite. The pre-built database file and associated datasets are included as part of the Jarvis distribution.

The front end for this demonstration is a Rich Internet Application using the powerful ExtJS widget set. The graphic design for this page is by Reality Software.

The demo pages show the use of the key features of Jarvis, including Create/Update/Delete operations on tables, plus examples of how to Login/Logout of the application, and how to use simple group-based security on both the server and client sides.
